# MindMeet

MindMeet is a comprehensive productivity and collaboration platform designed to help you organize thoughts, brainstorm ideas (your imagination is the limit for what you can create with the mindmaps) and connect with others. It combines powerful mind mapping capabilities with real-time chat, AI assistance, and social networking features.

## üåü Features

*   **Interactive Mindmaps:** Create dynamic mindmaps with various node types including text, images, links, and even media playlists.
*   **Real-time Collaboration:** Collaborate seamlessly with others on mind maps with live cursor tracking and updates.
*   **AI-Powered Assistance:** Leverage AI for brainstorming, content generation, and intelligent suggestions using integrated AI tools.
*   **Social Connectivity:** Share your creations, follow other users, and discover public mind maps through a personalized feed.
*   **Media Integration:** Embed and play content directly from Spotify and SoundCloud within your workspace.
*   **Secure Authentication:** Robust user management and authentication powered by Supabase.
*   **Modern Interface:** A sleek, responsive UI built with Tailwind CSS and Framer Motion for a smooth user experience.
